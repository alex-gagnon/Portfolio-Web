Server API Swagger Page

Quote APIs

  • /api/test

    Returns a test response

  • /api/v1/quotes

    Returns all quotes.

  • /api/v1/quotes/random

    Returns a random quote

  • /api/v1/quotes/quote/:id

    Returns a quote by id. change ":id" to a number.

Data APIs

  • /api/v1/data

    Returns all data
